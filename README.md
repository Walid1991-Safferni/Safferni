# سفّرني | Safferni

Intercity ride booking service — Syria, Amman & Beirut.

## Routes
- Damascus ↔ Amman
- Damascus ↔ Beirut
- Damascus ↔ Aleppo
- Damascus ↔ Homs

## Tech
- React 18 + Vite
- Bilingual (Arabic RTL + English LTR)
- Montserrat font

## Run locally
```
npm install
npm run dev
```

## Deploy
Push to GitHub → Import to Vercel → Done.
